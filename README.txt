Hello!

This project was based on the impossible game by grip digital which was the game that inspired
geometry dash. The goal of the project was to combine the tighter gameplay of the impossible
game with geometry dash's visuals and music.

Editor Controls:
LEFT CLICK : SELECT AND PLACE BLOCK
RIGHT CLICK/DRAG : PAN CAMERA

F1 : SAVE CURRENT LEVEL (levels can be found in the levels folder)
F2 : LOAD CURRENT SAVED LEVEL
F3 : START GAME (if you dont feel like making a level, hey we made one for you)

A : AMOGUS MODE (don't press)

Game Controls:
UARROW or SPACE : JUMP
R : PRINT TO CONSOLE CURRENT CAMERA POSITION (if you want to time lighting with music this is useful)
E : NOCLIP (cheater)

oh and don't die
